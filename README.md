## 🚀 Featured Project

### 🔐 GhostID — AI Facial Recognition Cloaking

A flagship privacy-tech project built using **Projected Gradient Descent (PGD)** and **FaceNet embeddings**.

This system applies **imperceptible adversarial noise** to facial images, making them:

* ✅ recognizable to humans
* ❌ unrecognizable to AI surveillance systems

### 📌 Key Achievement

Achieved **1.61 Euclidean distance**, significantly surpassing the **1.1 cloaking threshold**
