## 🔐 Flagship Project — GhostID

> *"What if your photo could be invisible to AI surveillance — but still look completely normal to humans?"*

**GhostID** is a privacy-enhancing system that applies imperceptible adversarial perturbations to facial images using **Projected Gradient Descent (PGD)**, making them mathematically invisible to facial recognition AI while remaining fully recognizable to human eyes.

```
Original Photo  ──[ PGD ε-noise ]──▶  Cloaked Photo
     │                                      │
     ▼                                      ▼
Human: ✓ Recognizes                  AI: ✗ No Match
```

| Metric | Result | Benchmark | Margin |
|--------|--------|-----------|--------|
| Euclidean Distance | **1.61** | 1.10 | **+46% above threshold** |
| Cloaking Success | ✅ | > 1.1 required | Achieved |

**Stack:** React.js · Flask · PyTorch · FaceNet · Node.js · MongoDB · PGD Algorithm · JWT Auth
